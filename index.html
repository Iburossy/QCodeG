Guide d'utilisation

1

Entrez votre URL

Saisissez l'adresse web que vous souhaitez encoder dans le champ URL.

2

Personnalisez votre QR code

Utilisez les onglets Style, Logo et Texte pour personnaliser l'apparence de votre QR code.

3

Générez le QR code

Cliquez sur le bouton Générer le QR Code pour créer votre QR code personnalisé.

4

Téléchargez votre QR code

Choisissez le format souhaité (PNG, JPEG, SVG, PDF) et téléchargez votre QR code.

Projet

QcodeG est un générateur de QR code personnalisable et gratuit.

Fonctionnalités :

  • Génération de QR codes à partir d'URLs
  • Personnalisation complète (couleurs, styles, taille)
  • Ajout de texte personnalisé
  • Téléchargement dans différents formats (PNG, JPEG, SVG, PDF)
